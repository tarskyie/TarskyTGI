TarskyTGI — a powerful and versatile native Windows application made to bring generative 
AI directly to your desktop.
  
Key Features:
  * Versatility: Run local, private AI models for offline access and complete data privacy, or connect to the powerful OpenAI API to leverage cutting-edge, cloud-based models.
  * Multiple Interaction Modes:
      * Chat: Engage in dynamic, back-and-forth conversations with your chosen AI.
      * Instruct: Provide direct instructions to the model for specific text generation tasks.
      * Host & Connect: Easily manage and host your own local models.
  * Native Windows Experience: Built as a modern Windows application, TarskyTGI offers the performance, stability, and
    familiar user experience you expect. It's not a web app in a wrapper.
  * Simple Setup: Get up and running in minutes with a straightforward installer. 

Who is this for?
  * Developers: Experiment with different models, test prompts, and integrate AI into your local development workflow
    with ease.
  * Writers & Creators: Break through creative blocks, generate ideas, draft content, and explore new writing styles in
    a private, secure environment.
  * AI Enthusiasts: Run and compare different open-source models directly on your own hardware.


TarskyTGI empowers you to harness the power of text generation AI with flexibility, privacy, and performance, all from your Windows PC.
